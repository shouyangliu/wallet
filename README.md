## 📱 这是什么

yl记账是一个个人记账工具。你可以用它来记录每天的收支，管理多个账户，查看消费统计图表。数据存在本地，也可以开启云端同步，换手机也不丢记录。

---

## ✨ 主要功能

| 功能 | 说明 |
|------|------|
| 🏦 **多账户** | 用颜色和 emoji 区分你的现金、银行卡、信用卡等多个账户 |
| 📝 **记账** | 快速记录每一笔开销和收入，支持分类和备注 |
| 📊 **统计图表** | 饼图展示消费分布，一眼看清钱花在哪 |
| 🔍 **筛选** | 按账户、类别、日期范围查找记录 |
| 📂 **导入/导出** | 支持 CSV 格式备份和迁移数据 |
| ☁️ **云端同步** | 登录后数据自动备份到云端，换设备不丢失 |
| 🌙 **深色模式** | 可以切换夜间模式 |
| 🎨 **主题配色** | 自由选择喜欢的颜色 |

---

## 📲 快速上手

### 安装应用

```bash
# 确保你有 Flutter 环境
flutter pub get
flutter run
```

### 开始记账

1. 添加你的账户（现金、银行卡等）
2. 点击 + 记录一笔收支
3. 月底查看图表统计

### 开启云端同步（可选）

在 App 设置页点击 **登录云端** → 注册账号 → 数据自动同步。

---

## 🙋 常见问题

**数据安全吗？**
数据默认存在你手机本地。如果开启云端同步，数据存在你自己的 Supabase 数据库里，只有你能访问。

**没网络能用吗？**
可以。App 默认纯本地运行，所有操作无需联网。
